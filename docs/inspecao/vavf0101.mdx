# VAVF0101 — Parâmetros Avaliação Fornecedores

> Painel de configuração dos parâmetros que regem a avaliação de fornecedores, com 3 sub-tabelas: Qualidade, Comercial e Logística.

## Objetivo

Permitir a definição e ponderação dos critérios utilizados na avaliação periódica de fornecedores. A tela e organizada em 3 sub-tabelas que correspondem as dimensões de avaliação: **Qualidade** (desempenho nas inspeções, ocorrencias, certificações), **Comercial** (preços, prazos de pagamento, flexibilidade negocial) e **Logística** (pontualidade de entrega, conformidade de volumes, documentação). Cada sub-tabela permite definir os critérios, seus pesos e a forma de cálculo, resultando em uma nota consolidada que alimenta o IQF (Indice de Qualidade do Fornecedor).

## Pre-requisitos

- [VAVR0200 (Fornecedores/Transportadoras)](/suprimento/vavr0200) — Fornecedores que serão avaliados com base nestes parâmetros.
- [VINS0105 (Tipos de Ocorrencias)](/inspecao/vins0105) — Tipos de ocorrencia que impactam a dimensão Qualidade.
- [VAVF0204 (Envio IQF Fornecedores)](/inspecao/vavf0204) — Tela que exibe o IQF calculado com base nestes parâmetros.

## Passo a passo

1. Acesse a tela VAVF0101 pelo menu Inspeção > Parâmetros Avaliação Fornecedores.
2. A tela exibe as **3 sub-tabelas** lado a lado ou em abas: Qualidade, Comercial e Logística.
3. Para cada sub-tabela, defina os **Critérios de Avaliação** que serão considerados.
4. Atribua um **Peso** (percentual) a cada criterio. A soma dos pesos dentro de cada sub-tabela deve totalizar 100%.
5. Defina a **Fórmula de Cálculo** para cada criterio (ex: média de notas, última nota, tendência).
6. Configure a **Periodicidade da Avaliação**: Mensal, Trimestral, Semestral ou Anual.
7. Defina as **Faixas de Classificação**: A (otimo), B (bom), C (regular), D (ruim) — com os intervalos de nota.
8. Configure o **Peso de Cada Dimensão** no cálculo final (ex: Qualidade 50%, Comercial 25%, Logística 25%).
9. Clique em **Salvar** para armazenar a configuração de parâmetros.
10. Utilize o botão **Simular** para testar a avaliação com dados de um fornecedor real antes de colocar em produção.

## Observações importantes

- A **soma dos pesos** das 3 dimensões deve totalizar 100%. O sistema valida esta condição e impede salvar se houver divergência.
- A **Fórmula Personalizada** permite escrever uma expressão SQL ou função customizada para calculos complexos, exigindo conhecimento técnico avancado.
- O botão **Simular** e uma ferramenta poderosa para calibrar os parâmetros. Ele aplica os critérios configurados sobre dados historicos de um fornecedor e mostra o resultado que seria obtido.
- As **Faixas de Classificação** determinam a cor e o tratamento do fornecedor: A (verde, recomendado), B (azul, aceitável), C (amarelo, requer atenção), D (vermelho, restrito).
- Fornecedores classificados como **D** por periodos consecutivos podem ser automaticamente bloqueados para novos pedidos de compra, conforme parametrização.
- A alteração dos parâmetros de avaliação **não retroage** — avaliações já realizadas mantém os critérios e pesos vigentes a epoca do cálculo.
