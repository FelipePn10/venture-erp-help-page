# VAVF0101 — Parametros Avaliacao Fornecedores

> Painel de configuracao dos parametros que regem a avaliacao de fornecedores, com 3 sub-tabelas: Qualidade, Comercial e Logistica.

## Objetivo

Permitir a definicao e ponderacao dos criterios utilizados na avaliacao periodica de fornecedores. A tela e organizada em 3 sub-tabelas que correspondem as dimensoes de avaliacao: **Qualidade** (desempenho nas inspecoes, ocorrencias, certificacoes), **Comercial** (precos, prazos de pagamento, flexibilidade negocial) e **Logistica** (pontualidade de entrega, conformidade de volumes, documentacao). Cada sub-tabela permite definir os criterios, seus pesos e a forma de calculo, resultando em uma nota consolidada que alimenta o IQF (Indice de Qualidade do Fornecedor).

## Pre-requisitos

- [VAVR0200 (Fornecedores/Transportadoras)](/suprimento/vavr0200) — Fornecedores que serao avaliados com base nestes parametros.
- [VINS0105 (Tipos de Ocorrencias)](/inspecao/vins0105) — Tipos de ocorrencia que impactam a dimensao Qualidade.
- [VAVF0204 (Envio IQF Fornecedores)](/inspecao/vavf0204) — Tela que exibe o IQF calculado com base nestes parametros.

## Passo a passo

1. Acesse a tela VAVF0101 pelo menu Inspecao > Parametros Avaliacao Fornecedores.
2. A tela exibe as **3 sub-tabelas** lado a lado ou em abas: Qualidade, Comercial e Logistica.
3. Para cada sub-tabela, defina os **Criterios de Avaliacao** que serao considerados.
4. Atribua um **Peso** (percentual) a cada criterio. A soma dos pesos dentro de cada sub-tabela deve totalizar 100%.
5. Defina a **Formula de Calculo** para cada criterio (ex: media de notas, ultima nota, tendencia).
6. Configure a **Periodicidade da Avaliacao**: Mensal, Trimestral, Semestral ou Anual.
7. Defina as **Faixas de Classificacao**: A (otimo), B (bom), C (regular), D (ruim) — com os intervalos de nota.
8. Configure o **Peso de Cada Dimensao** no calculo final (ex: Qualidade 50%, Comercial 25%, Logistica 25%).
9. Clique em **Salvar** para armazenar a configuracao de parametros.
10. Utilize o botao **Simular** para testar a avaliacao com dados de um fornecedor real antes de colocar em producao.

## Observacoes importantes

- A **soma dos pesos** das 3 dimensoes deve totalizar 100%. O sistema valida esta condicao e impede salvar se houver divergencia.
- A **Formula Personalizada** permite escrever uma expressao SQL ou funcao customizada para calculos complexos, exigindo conhecimento tecnico avancado.
- O botao **Simular** e uma ferramenta poderosa para calibrar os parametros. Ele aplica os criterios configurados sobre dados historicos de um fornecedor e mostra o resultado que seria obtido.
- As **Faixas de Classificacao** determinam a cor e o tratamento do fornecedor: A (verde, recomendado), B (azul, aceitavel), C (amarelo, requer atencao), D (vermelho, restrito).
- Fornecedores classificados como **D** por periodos consecutivos podem ser automaticamente bloqueados para novos pedidos de compra, conforme parametrizacao.
- A alteracao dos parametros de avaliacao **nao retroage** — avaliacoes ja realizadas mantem os criterios e pesos vigentes a epoca do calculo.
