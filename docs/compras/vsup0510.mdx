# VSUP0510 — Apoio de Fornecedores

> Manter os cadastros de apoio do fornecedor: **Tipos de Fornecedor** (com o `kind` que controla a obrigatoriedade da IE), **Tipos de Contato** e os **Parâmetros por empresa**.

## Objetivo

Fornecer as tabelas de apoio consumidas pelo cadastro de fornecedor (VSUP0500) e configurar o comportamento do módulo por empresa.

## Passo a passo

1. Aba **Tipos**: cadastre tipos de fornecedor informando descrição e **kind** (Normal, Transportadora, Transp. Redespacho, Redespacho).
2. Aba **Contatos**: cadastre os tipos de contato (Comprador, Gerente…).
3. Aba **Parâmetros**: por empresa, defina conta financeira default, se exige conta, homologação default, **data base padrão para vencimentos** (Emissão/Entrada/Digitação), fornecedor genérico da NF-e, etc.

## Telas relacionadas

- **[VSUP0500 (Cadastro de Fornecedor)](/compras/vsup0500):** consome estes apoios.
