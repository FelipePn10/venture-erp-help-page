# VPRE0201 — Cadastro Previsão Vendas

> Cadastro de previsões de venda com alternância entre visões mensal e semanal, permitindo planejamento detalhado da demanda futura.

## Objetivo

Permitir o cadastro e manutenção das previsões de venda que alimentam o planejamento de produção e suprimentos. A tela oferece duas visões complementares: **Mensal** (para planejamento de médio e longo prazo, tipicamente 12 a 18 meses) e **Semanal** (para planejamento de curto prazo, tipicamente 4 a 13 semanas). As previsões podem ser inseridas manualmente ou importadas de arquivos externos, e servem como principal entrada para o processo de S&OP e MRP.

## Pre-requisitos

- [VPRE0101 (Tabela Apropriação)](/previsao/vpre0101) — Tabelas de apropriação para rateio das previsões.
- [VPRE0102 (Bloqueio Previsão)](/previsao/vpre0102) — Configuração de semanas bloqueadas para edição.
- [VENT0200 (Cadastro de Itens)](/engenharia/vent0200) — Itens que serão objeto das previsões de venda.

## Passo a passo

1. Acesse a tela VPRE0201 pelo menu Previsão > Cadastro Previsão Vendas.
2. Selecione o **Ano** de referência para a previsão.
3. Escolha a **Visão Mensal** ou **Visão Semanal** no seletor de visualização.
4. Selecione o **Item** ou **Familia de Produtos** para o qual deseja cadastrar a previsão.
5. Na **Visão Mensal**, preencha a quantidade prevista para cada mês do ano. O sistema exibe o total anual automaticamente.
6. Na **Visão Semanal**, preencha a quantidade prevista para cada semana. A visão semanal permite maior granularidade para itens com alta rotatividade.
7. Utilize o botão **Ratear** para distribuir um total anual entre os meses ou semanas, utilizando uma tabela de sazonalidade ou percentuais manuais.
8. Selecione a **Tabela de Apropriação** para rateio automático entre centros de custo.
9. Para **Importar** previsões, utilize o botão **Importar Planilha** e selecione o arquivo Excel com os dados.
10. Clique em **Salvar** para armazenar a previsão.

## Campos

| Campo | Tipo | Obrigatório | Opções | Descrição |
|-------|------|-------------|--------|-----------|
| Ano | select | Sim | Lista de anos | Ano de referência da previsão |
| Visão | select | Sim | Mensal / Semanal | Granularidade da previsão |
| Item | select | Não | Lista de itens | Item específico da previsão |
| Familia de Produtos | select | Não | Lista de familias | Familia de produtos da previsão |
| Datas (meses/semanas) | cabeçalho | — | Gerado automaticamente | Periodos da previsão conforme ano e visão |
| Quantidade Prevista | número | Sim | — | Quantidade prevista para cada período |
| Total Anual | número | Não | — | Calculado automaticamente: soma de todos os periodos |
| Tabela de Apropriação | select | Não | Lista de tabelas | Tabela para rateio entre centros de custo |
| Tabela de Sazonalidade | select | Não | Lista de tabelas | Tabela para distribuição sazonal automática |
| Status | select | Não | Em Edição / Consolidado / Congelado | Status da previsão |

## Observações importantes

- A **Visão Semanal** e automaticamente derivada da visão mensal quando não preenchida manualmente, utilizando rateio proporcional pelos dias úteis de cada semana.
- O **Ratear** com tabela de sazonalidade e útil para itens com padrão sazonal conhecido (ex: sorvetes vendem mais no verão), aplicando percentuais historicos automaticamente.
- Previsões com status **Consolidado** podem ser alteradas apenas por usuários com permissão específica. Status **Congelado** bloqueia totalmente as alterações.
- A **Importação de Planilha** suporta formatos pre-definidos. Baixe o modelo de planilha na tela antes de preparar os dados para garantir compatibilidade.
- As previsões são a principal entrada para o **MRP**. Previsões imprecisas geram distorções em toda a cadeia: excesso de estoque ou ruptura de produção.
- E recomendável realizar ciclos regulares de revisão das previsões, tipicamente mensais, comparando o previsto com o realizado e ajustando as projeções.
