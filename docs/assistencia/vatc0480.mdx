# VATC0480 — Consulta Chamados

> Tela de consulta de chamados de assistência técnica com 12 filtros e processamento client-side que oferece resposta instantanea as buscas.

## Objetivo

Fornecer uma interface de consulta agil aos chamados de assistência técnica, utilizando processamento **client-side** (no navegador do usuário) para oferecer resposta instantanea as buscas. Diferentemente das consultas tradicionais que fazem requisições ao servidor a cada mudança de filtro, esta tela carrega um conjunto de dados e aplica os filtros localmente, resultando em tempo de resposta praticamente nulo. E ideal para usuários que precisam navegar rapidamente entre diferentes filtros e cenários de busca.

## Pre-requisitos

- [VASS0201 (Cadastro Chamado Assistência)](/assistencia/vass0201) — Chamados que compõem a base de dados.
- [VATC0280 (Cadastro Chamados Alternativo)](/assistencia/vatc0280) — Chamados de vistoria também consultados.
- [VASS0402 (Consulta Assistência)](/assistencia/vass0402) — Consulta tradicional para buscas que exigem maior volume de dados.

## Passo a passo

1. Acesse a tela VATC0480 pelo menu Assistência > Consulta Chamados.
2. Ao abrir, a tela carrega os chamados do período padrão (tipicamente ultimos 30 dias) no navegador.
3. A barra de **Busca Rápida** no topo aceita número do chamado, nome do cliente ou número de série e filtra instantaneamente a cada tecla digitada.
4. Utilize os **12 filtros** na barra lateral para refinar os resultados:
   - **Filtro 1 — Tipo de Chamado**: 6 checkboxes (Garantia, Orçamento, Revisão, Recall, Cortesia, Interno).
   - **Filtro 2 — Status**: checkboxes para cada status.
   - **Filtro 3 — Técnico**: dropdown com busca textual.
   - **Filtro 4 — Produto**: dropdown com busca textual.
   - **Filtro 5 — Cliente**: campo de texto com autocomplete.
   - **Filtro 6 — Região**: dropdown de UFs.
   - **Filtro 7 — Cobertura de Garantia**: toggle Dentro/Fora.
   - **Filtro 8 — Data de Abertura**: seletor de período.
   - **Filtro 9 — Data de Conclusão**: seletor de período.
   - **Filtro 10 — Valor**: controle deslizante (slider) de faixa de valor.
   - **Filtro 11 — Número de Série**: campo de texto.
   - **Filtro 12 — Atrasados**: toggle para filtrar apenas chamados com SLA vencido.
5. Todos os filtros são aplicados **instantaneamente** (client-side), sem recarregar a pagina.
6. A grid de resultados atualiza em tempo real conforme os filtros são alterados.
7. O **contador de resultados** mostra quantos chamados estao sendo exibidos do total carregado.
8. Para aumentar a base de dados, utilize o seletor de **Período de Carga** para trazer mais meses ao navegador.
9. Clique em um chamado para abrir o **modal de detalhamento rapido** com as principais informações.
10. Utilize o botão **Exportar Grid** para gerar Excel apenas com os registros filtrados no momento.

## Campos de Filtro

| Campo | Tipo | Obrigatório | Opções | Descrição |
|-------|------|-------------|--------|-----------|
| 1. Tipo de Chamado | checkboxes | Não | 6 tipos | Seleção múltipla via checkboxes |
| 2. Status | checkboxes | Não | Todos os status | Seleção múltipla via checkboxes |
| 3. Técnico | dropdown | Não | Lista de técnicos | Busca textual com filtro |
| 4. Produto | dropdown | Não | Lista de itens | Busca textual com filtro |
| 5. Cliente | texto | Não | — | Autocomplete enquanto digita |
| 6. Região | dropdown | Não | Lista de UFs | Seleção única |
| 7. Cobertura de Garantia | toggle | Não | Dentro / Fora / Todos | Toggle de 3 estados |
| 8. Data Abertura | período | Não | — | Seletor de data inicial e final |
| 9. Data Conclusão | período | Não | — | Seletor de data inicial e final |
| 10. Valor | slider | Não | Min-Max | Controle deslizante de faixa |
| 11. Número de Série | texto | Não | — | Busca textual |
| 12. Atrasados (SLA Vencido) | toggle | Não | — | Filtra apenas chamados com prazo expirado |

## Observações importantes

- O processamento **client-side** e o grande diferencial desta tela. Como os dados já estao no navegador, a aplicação dos filtros e instantanea, proporcionando uma experiência fluida de análise exploratoria.
- A **limite da base client-side** e tipicamente configurado para 3 a 6 meses de dados, equilibrando performance e abrangência. Para periodos maiores, utilize a tela [VASS0402 (Consulta Assistência)](/assistencia/vass0402).
- A **Busca Rápida** e extremamente eficiente para localizar um chamado específico pelo número, nome do cliente ou número de série, com resultados em tempo real a cada tecla.
- O **controle deslizante de valor** (slider) e intuitivo para definir faixas de preço, substituindo a digitação manual de valores mínimo e máximo.
- O modal de **detalhamento rapido** exibe as informações principais sem recarregar a pagina, mantendo o contexto da consulta.
- Para recarregar a base com dados mais recentes (ex: incluir chamados do dia), utilize o botão **Atualizar Dados** que busca incrementalmente no servidor apenas os novos registros.
