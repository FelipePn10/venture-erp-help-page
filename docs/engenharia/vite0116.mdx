# VITE0116 — Atributos PDM

> Cadastro dos atributos (valores) de cada modificador PDM, com funcionalidades avancadas de Copiar e Colar entre modificadores diferentes.

## Objetivo

Permitir o cadastro dos atributos PDM, que representam os valores possíveis para cada modificador. Por exemplo, para o modificador "Cor", os atributos seriam "Azul", "Vermelho", "Preto", etc. A tela oferece uma funcionalidade exclusiva de **Copiar/Colar** atributos entre modificadores diferentes, extremamente útil quando dois modificadores compartilham os mesmos valores (ex: "Cor Externa" e "Cor Interna" podem ter os mesmos atributos de cor). Além disso, a tela permite definir propriedades específicas de cada atributo, como impacto na estrutura, custo adicional e disponibilidade.

## Pre-requisitos

- [VITE0115 (Modificadores PDM)](/engenharia/vite0115) — Os modificadores para os quais os atributos serão cadastrados devem existir.
- [VENT0204 (Cadastro de Grupo PDM)](/engenharia/vent0204) — Contexto dos grupos PDM onde os atributos serão utilizados.

## Passo a passo

1. Acesse a tela VITE0116 pelo menu Engenharia > Atributos PDM.
2. Selecione o **Modificador** para o qual deseja cadastrar ou visualizar atributos.
3. Para cadastrar um novo atributo, clique em **Novo** e preencha o código e a descrição do atributo.
4. Defina a **Ordem** do atributo dentro do modificador para controlar a posição em listas de seleção.
5. Se o atributo gerar um **Item Específico** na BOM, informe o código do item no campo correspondente.
6. Se houver **Custo Adicional**, informe o valor no campo apropriado.
7. Para **Copiar atributos** de outro modificador, clique em **Copiar**, selecione o modificador de origem e confirme.
8. Para **Colar atributos** copiados, clique em **Colar** após selecionar o modificador de destino.
9. Utilize os toggles de **Ativo** e **Disponível** para controlar quais atributos aparecem no configurador.
10. Clique em **Salvar** para persistir os atributos cadastrados.

## Campos

| Campo | Tipo | Obrigatório | Opções | Descrição |
|-------|------|-------------|--------|-----------|
| Modificador | select | Sim | Lista de modificadores | Modificador PDM ao qual o atributo pertence |
| Código do Atributo | texto | Sim | — | Identificador único do atributo |
| Descrição | texto | Sim | — | Nome descritivo do atributo (ex: "Azul Royal") |
| Ordem | número | Não | — | Posição do atributo na lista de seleção |
| Item Específico | select | Não | Lista de itens | Item gerado na BOM quando este atributo e selecionado |
| Custo Adicional | número | Não | — | Acréscimo de custo quando o atributo e escolhido |
| Ativo | toggle | Sim | — | Status do atributo no sistema |
| Disponível | toggle | Não | — | Disponibilidade do atributo no configurador |
| Atributo Padrão | toggle | Não | — | Define este atributo como opção padrão do modificador |
| Cor (Swatch) | cor | Não | — | Representação visual de cor para atributos de cor |
| Imagem | arquivo | Não | — | Imagem ilustrativa do atributo para o configurador visual |

## Observações importantes

- A funcionalidade de **Copiar/Colar** atributos entre modificadores e um grande diferencial de produtividade. Ao cadastrar "Cor Externa" e "Cor Interna", basta cadastrar os atributos em um modificador e cópia-los para o outro.
- O campo **Item Específico** e utilizado quando a escolha de um atributo demanda um componente diferente na estrutura BOM. Por exemplo, escolher "Vidro Temperado" (vs. "Vidro Comum") insere automaticamente o item de vidro temperado na estrutura.
- O **Custo Adicional** e somado ao custo base do produto, permitindo precificação diferenciada conforme as escolhas do cliente.
- A **Cor (Swatch)** e a **Imagem** enriquecem a experiência do configurador visual, exibindo uma amostra de cor ou foto do atributo no momento da seleção.
- Atributos marcados como **Não Disponível** permanecem cadastrados mas não aparecem no configurador, útil para gerenciar atributos sazonais ou em fim de linha.
- O **Atributo Padrão** e o que aparece pre-selecionado no configurador. Deve ser o mais comum ou o de menor custo para não induzir o cliente a uma escolha mais cara inadvertidamente.
