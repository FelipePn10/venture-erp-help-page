# VITE0114 — Grupos PDM (Tema Azul)

> Interface de visualização e navegação dos grupos PDM no tema azul, oferecendo uma experiência consolidada para gerenciamento da hierarquia de configuração.

## Objetivo

Fornecer uma visão consolidada e amigável de todos os grupos PDM cadastrados no sistema, utilizando o tema visual azul caracteristico do módulo de engenharia. Esta tela serve como painel central para navegação na hierarquia PDM, permitindo que o usuário visualize grupos, modificadores vinculados e atributos em uma única interface integrada. A partir desta tela, e possível navegar rapidamente para as telas de detalhe de modificadores, atributos e regras, funcionando como um hub de configuração PDM.

## Pre-requisitos

- [VENT0204 (Cadastro de Grupo PDM)](/engenharia/vent0204) — Grupos PDM devem estar previamente cadastrados.
- [VITE0115 (Modificadores PDM)](/engenharia/vite0115) — Modificadores vinculados aos grupos PDM.
- [VITE0116 (Atributos PDM)](/engenharia/vite0116) — Atributos associados aos modificadores de cada grupo.

## Passo a passo

1. Acesse a tela VITE0114 pelo menu Engenharia > Grupos PDM.
2. A tela exibe todos os grupos PDM cadastrados em formato de tabela com tema azul.
3. Utilize a **Busca** para localizar um grupo específico por código ou descrição.
4. Clique em um grupo para **expandir** e visualizar os modificadores PDM vinculados a ele.
5. Clique em um modificador expandido para visualizar os **atributos** associados.
6. Utilize o botão **Novo Grupo** para criar um novo grupo PDM, sendo redirecionado para a tela [VENT0204 (Cadastro de Grupo PDM)](/engenharia/vent0204).
7. Utilize o botão **Editar** para modificar um grupo existente.
8. O menu de contexto (clique direito) em um grupo oferece atalhos para Modificadores, Atributos e Regras do grupo selecionado.
9. Utilize o botão **Exportar** para gerar uma planilha com a hierarquia PDM completa.
10. O indicador visual de **Status** (bolinha verde/vermelha) mostra se o grupo possui configuração completa ou pendente.

## Campos

| Campo | Tipo | Obrigatório | Opções | Descrição |
|-------|------|-------------|--------|-----------|
| Busca | texto | Não | — | Campo de filtro por código ou descrição do grupo |
| Código do Grupo | texto | Não | — | Identificador do grupo PDM |
| Descrição | texto | Não | — | Nome descritivo do grupo PDM |
| Modificadores Vinculados | número | Não | — | Quantidade de modificadores associados ao grupo |
| Itens Gerados | número | Não | — | Quantidade de itens configurados gerados a partir do grupo |
| Status | indicador | Não | Completo / Pendente | Indicador visual da completude da configuração |
| Máscara | texto | Não | — | Máscara de codificação definida para o grupo |

## Observações importantes

- O **tema azul** identifica visualmente que o usuário esta no contexto do módulo de configuração PDM (Product Data Management).
- O indicador de **Status Pendente** (bolinha vermelha) sugere que o grupo PDM ainda não possui todos os elementos necessários (modificadores, atributos ou regras) para geração de itens configurados.
- A navegação hierárquica (Grupo > Modificador > Atributo) na própria tela evita que o usuário precise abrir múltiplas telas para visualizar a configuração completa.
- O menu de contexto oferece **atalhos rapidos** que agilizam o fluxo de trabalho, especialmente útil para consultores e analistas de engenharia.
- A exportação da hierarquia completa e útil para documentação e auditoria da configuração de produtos.
- Grupos com muitos itens gerados podem ter sua expansão mais lenta. Utilize a busca para localizar grupos específicos.
