# VENT0115 — Roteiro Padrão

> Templates de roteiros de fabricação pre-definidos que podem ser reutilizados como base para criação de roteiros de itens similares.

## Objetivo

Permitir o cadastro de roteiros de fabricação padrão que servem como templates para a criação de roteiros específicos de itens. Roteiros padrão são especialmente úteis em ambientes onde muitos itens compartilham a mesma sequência de operações, variando apenas em parâmetros específicos como tempos ou centros de trabalho. Ao utilizar um roteiro padrão como base, o usuário evita retrabalho na digitação das operações e garante consistência entre roteiros de itens similares.

## Pre-requisitos

- [VMAQ0101 (Tipos de Maquina)](/engenharia/vmaq0101) — Tipos de maquina para associação as operações do roteiro padrão.
- [VENT0202 (Roteiro de Fabricação)](/engenharia/vent0202) — Roteiros de fabricação que utilizarão o roteiro padrão como template.

## Passo a passo

1. Acesse a tela VENT0115 pelo menu Engenharia > Roteiro Padrão.
2. Clique em **Novo** para iniciar o cadastro de um novo roteiro padrão.
3. Preencha o **Código do Roteiro** e a **Descrição** que identifiquem claramente o tipo de processo (ex: "RT-MONTAGEM" para roteiro de montagem padrão).
4. Para cada operação do roteiro padrão, preencha a sequência, descrição e centro de trabalho.
5. Defina as **Formulas de Tempo** genéricas, utilizando variáveis como `QTD` para quantidade, que serão ajustadas posteriormente no roteiro específico.
6. Configure o **Tipo de Mao de Obra** e a **Quantidade de Operadores** padrão para cada operação.
7. Se aplicável, defina a **Taxa de Ocupação** padrão esperada para cada centro de trabalho.
8. Ordene as operações pela **Sequência** correta de execução utilizando os botões de reordenação se necessário.
9. Clique em **Salvar** para armazenar o template de roteiro padrão.

## Campos

| Campo | Tipo | Obrigatório | Opções | Descrição |
|-------|------|-------------|--------|-----------|
| Código do Roteiro | texto | Sim | — | Identificador único do roteiro padrão |
| Descrição | texto | Sim | — | Nome descritivo do roteiro padrão |
| Situação | select | Sim | Ativo / Inativo | Status do roteiro padrão |
| Sequência | número | Sim | — | Ordem da operação no roteiro padrão |
| Descrição da Operação | texto | Sim | — | Descrição detalhada da operação no template |
| Centro de Trabalho | select | Sim | Lista de centros | Centro de trabalho sugerido para a operação |
| Fórmula Tempo Setup | texto | Não | Expressão matemática | Fórmula padrão para tempo de preparação |
| Fórmula Tempo Execução | texto | Sim | Expressão matemática | Fórmula padrão para tempo de execução |
| Fórmula Tempo Finalização | texto | Não | Expressão matemática | Fórmula padrão para tempo de finalização |
| Tipo de Mao de Obra | select | Não | Lista de tipos de MO | Tipo de mao de obra sugerido |
| Quantidade de Operadores | número | Não | — | Número padrão de operadores |
| Taxa de Ocupação | número | Não | — | Taxa de ocupação sugerida |

## Observações importantes

- Roteiros padrão são apenas **templates** e não estao diretamente vinculados a nenhum item. Eles servem exclusivamente como base para cópia na tela de [VENT0202 (Roteiro de Fabricação)](/engenharia/vent0202).
- As formulas de tempo devem ser escritas de forma genérica, utilizando `QTD` como variável de quantidade, permitindo que sejam ajustadas no momento da cópia para o item específico.
- E recomendável criar roteiros padrão para cada **familia de produtos** com processos produtivos similares, reduzindo o tempo de cadastro de novos itens.
- A alteração de um roteiro padrão **não afeta** automaticamente os roteiros já copiados para itens. Para propagar mudancas, e necessário recopiar o roteiro padrão.
- Mantenha uma nomenclatura padronizada para os códigos de roteiros padrão, facilitando a localização e identificação por outros usuários da engenharia.
- Roteiros padrão inativos não aparecem na lista de seleção durante a cópia na tela de roteiro de fabricação.
