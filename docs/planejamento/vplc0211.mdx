# VPLC0211 — Orientações Entrega

> Cadastro de orientações de entrega com recurso de auto-preenchimento de endereço via CEP, facilitando a logística de expedição.

## Objetivo

Permitir o cadastro e manutenção das orientações de entrega associadas aos pedidos de venda, definindo rotas, endereços de entrega e instruções logísticas. A tela oferece um recurso de **auto-preenchimento de endereço a partir do CEP**, que consulta uma base de CEPs e preenche automaticamente logradouro, bairro, cidade e estado, reduzindo erros de digitação e agilizando o processo. As orientações de entrega são utilizadas pela tela de montagem de carga para agrupar pedidos com rotas compatíveis.

## Pre-requisitos

- [VPLC0200 (Montagem de Carga)](/planejamento/vplc0200) — Montagem de carga que consome as orientações de entrega.
- [VAVR0200 (Fornecedores/Transportadoras)](/suprimento/vavr0200) — Transportadoras disponíveis para as rotas.
- [Cadastro de Clientes](/cliente/vcli0500) — Clientes com endereços de entrega cadastrados.

## Passo a passo

1. Acesse a tela VPLC0211 pelo menu Planejamento > Orientações Entrega.
2. Clique em **Nova** para criar uma nova orientação de entrega.
3. Selecione o **Pedido de Venda** ao qual a orientação sera vinculada.
4. Informe o **CEP** de entrega. O sistema consultara automaticamente a base de CEPs e preenchera os campos de endereço.
5. Verifique os dados preenchidos automaticamente: **Logradouro**, **Bairro**, **Cidade**, **Estado**. Corrija manualmente se necessário.
6. Complete o endereço com o **Número** e **Complemento**.
7. Selecione a **Rota de Entrega** que define o trajeto logístico.
8. Escolha a **Transportadora** responsável pela entrega.
9. Informe instruções adicionais no campo **Observação** (ex: "Entregar no almoxarifado B", "Agendar horário com antecedência").
10. Clique em **Salvar** para registrar a orientação de entrega.

## Observações importantes

- O **auto-preenchimento por CEP** e uma funcionalidade que reduz significativamente erros de endereço. O sistema consulta a base de CEPs e preenche os campos de endereço automaticamente em menos de 1 segundo.
- Caso o CEP não seja encontrado na base, os campos permanecerao vazios e o usuário devera preenche-los manualmente. CEPs não encontrados são registrados para atualização futura da base.
- A **Rota de Entrega** e utilizada pela tela de [VPLC0200 (Montagem de Carga)](/planejamento/vplc0200) para agrupar pedidos que seguem o mesmo trajeto logístico, otimizando a consolidação de cargas.
- A **Janela de Entrega** e uma informação importante para o planejamento logístico, especialmente em centros urbanos com restrições de horário para circulação de veículos de carga.
- E possível alterar as orientações de entrega mesmo após o pedido estar liberado, desde que a carga ainda não tenha sido fechada na montagem de carga.
- Orientações com status **Cancelada** são desconsideradas pela montagem de carga, mas permanecem registradas para auditoria.
