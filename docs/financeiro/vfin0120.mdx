# VFIN0120 — Plano de Contas

> Manter o plano de contas contábil da empresa com notação hierárquica.

**Código fonte:** `src/components/screens/financeiro/Vfin0120Page.tsx`

## Objetivo

Manter o plano de contas contábil da empresa. Utiliza notação hierárquica com pontos para representar a estrutura de arvore contábil. Serve como classificação contábil para os títulos financeiros e como base para o relatório DRE (R05).

## Pre-requisitos

- Conhecimento básico de estrutura contábil (grupos, subgrupos, contas analíticas).
- Recomenda-se definir a estrutura hierárquica antes de iniciar o cadastro.

## Passo a passo

1. Acesse **Financeiro > Plano de Contas** (VFIN0120).
2. Para nível 1 (grupo): **+ Nova Conta**, informe Código simples (ex: `1`, `2`, `3`), Descrição, Tipo e Natureza. Deixe Código Pai em branco.
3. Para níveis inferiores: informe Código com ponto (ex: `3.1`, `3.1.01`), preencha Código Pai com a conta superior.
4. Clique em **Salvar**.

## Campos

| Campo | Tipo | Obrigatório | Opções | Função |
|-------|------|-------------|--------|--------|
| Código | texto | Sim | — | Código hierárquico com notação de ponto (ex: `3.1.01`) |
| Descrição | texto | Sim | — | Nome descritivo da conta |
| Código Pai | texto | Não | — | Código da conta superior na hierarquia |
| Tipo | seleção | Sim | RECEITA / DESPESA / ATIVO / PASSIVO / PATRIMONIO | Classificação contábil |
| Natureza | seleção | Sim | CRÉDITO / DÉBITO | Natureza do saldo |

## Observações importantes

- O nível e calculado automaticamente pela quantidade de segmentos no código.
- O plano de contas e a base para o relatório R05 (DRE) no VFIN0500.
- Alterações após início da operação devem ser feitas com cautela.
